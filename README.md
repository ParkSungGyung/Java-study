# ✅컴퓨터 견적 리스트 프로그램
---------------------------------------------------------------------------

> ## ✅프로젝트 개요
> 1. 목적 : 컴퓨터의 각 부품 별 리스트를 이용하여 자신만의 컴퓨터 견적을 조합해 총 합과 사양, 호환성 검사등을 할 수 있는 프로그램입니다.
>
>
>
> 2. 주요 기능 
> |사용자|기능|
  |---|---|
  |유저|컴퓨터 부품 조회, 견적 생성, 호환성 검사|
  |관리자|컴퓨터 부품 리스트 CRUD, 유저 견적 조회, 수정|
